## Common Mistakes to Avoid

| Mistake | Why It's Wrong | Correct Approach |
|---------|----------------|------------------|
| Using `#FFFFFF` white background | Clashes with warm palette | Use `neutral-100` (cream) |
| Using `#000000` for text | Too harsh, not brand | Use `neutral-900` (warm dark) |
| Using cold gray `#808080` | Breaks warm aesthetic | Use `neutral-500` (warm taupe) |
| Primary button with dark text | Poor contrast | Always white text on primary |
| Red for decorative elements | Confuses with errors | Reserve red only for errors |
| Terracotta for main CTA | Dilutes brand hierarchy | Primary CTA = coffee brown |

## Usage Guidelines

### Do's ‚úÖ

- **Do** use the warm cream background as your default - it's the brand signature
- **Do** use coffee brown (`primary-500`) for primary CTAs and key actions
- **Do** use terracotta (`secondary-500`) for secondary elements and visual interest
- **Do** maintain the warm feeling throughout - avoid cold grays
- **Do** use the full scale (50-900) for subtle variations and depth
- **Do** check contrast using the Magic Number quick reference

### Don'ts ‚ùå

- **Don't** use cold gray neutrals - they clash with the warm brand colors
- **Don't** overuse the accent (burgundy) - reserve for emphasis
- **Don't** place light text on medium backgrounds without checking contrast
- **Don't** use primary colors for error states (reserve red for errors)
- **Don't** ignore the 60-30-10 rule - too much color creates visual chaos
- **Don't** modify the locked brand colors - they are the source of truth
