## Uso

Cada offer sheet é **autossuficiente**. Pode ser compartilhado individualmente com prospects após o diagnóstico.

**Fluxo recomendado:**
1. Diagnóstico gratuito
2. Identificar serviço(s) relevante(s)
3. Enviar offer sheet específico
4. Proposta personalizada

---

## Estrutura de Cada Offer Sheet

- O Problema (dor que resolve)
- O Que É (descrição do serviço)
- O Que Inclui (entregas)
- Pra Quem É / Não É (qualificação)
- Suas Responsabilidades (o que o cliente faz)
- Como Medimos Sucesso (métricas anti-vaidade)
- Compromisso (período, pagamento)
- Próximo Passo (CTA)
- Combina Com (cross-sell natural)
