The signature gradient from the brand manual.


**OKLCH version:**
```css
background: linear-gradient(135deg,
  oklch(30% 0.110 25) 0%,
  oklch(38% 0.145 28) 50%,
```

The palette follows the Magic Number system where same grades across color families have the same OKLCH Lightness (within 2%), enabling predictable contrast calculations.

### Quick Reference

- **Text on light backgrounds:** Use grade 600+ (minimum AA)
- **Text on dark backgrounds:** Use grade 300 or lighter
- **Interactive elements:** Ensure 40+ grade difference from background
- **Large text/UI components:** Minimum 30 grade difference

## Usage Guidelines

### 60-30-10 Rule

| Proportion | Color | Application |
|------------|-------|-------------|
| 60% | Neutrals (Caramelo, Cinza) | Backgrounds, large surfaces |
| 30% | Primary/Secondary | Headers, navigation, key UI |
| 10% | Accent (Terracotta) | CTAs, highlights, emphasis |

### Do's

- Use the brand gradient for hero sections and prominent brand areas
- Pair cream backgrounds (neutral-50/100) with dark text (gray-900)
- Use terracotta accent sparingly for maximum impact
- Apply dark mode tokens consistently across the interface
- Maintain minimum 40+ grade difference for text/background pairs

### Don'ts

- Don't use primary-700 for large text blocks (too heavy)
- Don't combine multiple saturated colors at full intensity
- Don't use gray-400/500 for important text (insufficient contrast)
- Don't place semantic colors (success/error) without context
- Don't modify locked colors - they are brand-definitive

## Implementation

### CSS Custom Properties

Import the CSS file for immediate use:

```html
<link rel="stylesheet" href="oitavo-cafe-colors.css">
```

Then use semantic tokens:

```css
.my-component {
  background: var(--bg-default);
  color: var(--text-default);
  border: 1px solid var(--border-default);
}

.my-button {
  background: var(--interactive-default);
  color: var(--text-inverse);
}

.my-button:hover {
  background: var(--interactive-hover);
}
```

### Dark Mode Toggle

Add `.dark` class to root element:

```javascript
document.body.classList.toggle('dark');
```

Or use `data-theme` attribute:

```html
<html data-theme="dark">
```

### Direct Color Usage

When semantic tokens don't fit, use scale values:

```css
.special-element {
  background: var(--color-accent-100);
  border-left: 4px solid var(--color-accent-500);
}
```
