# Idea Refresh - Validation Checklist

## Context Validation
- [ ] Previous research status confirmed (available or minimal mode)
- [ ] Niche identified
- [ ] Refresh focus documented
- [ ] If previous research used: date of original research noted

## Trend Update Quality
- [ ] Quick trend scan completed
- [ ] Major trend shifts identified (or confirmed stable)
- [ ] New trending elements documented if applicable

## Fresh Ideas Quality
- [ ] Exactly {{idea_count}} ideas generated
- [ ] Each idea has all required components:
  - [ ] Catchy title
  - [ ] Scroll-stopping hook
  - [ ] Clear core concept
  - [ ] Specific format
  - [ ] Trending element
  - [ ] Viral potential factors
  - [ ] Instagram optimization strategy
- [ ] Ideas are DIFFERENT from previous batch (if applicable)
- [ ] Refresh focus directive was applied:
  - [ ] Different angles (if requested)
  - [ ] Different formats (if requested)
  - [ ] Trending variations (if requested)
  - [ ] All new (if requested)
- [ ] Ideas show variety across content types and formats

## Ranking Quality
- [ ] All {{idea_count}} ideas scored and ranked
- [ ] Each idea has viral probability score (1-100)
- [ ] Clear ranking (#1 to #{{idea_count}})
- [ ] Strategic rationale provided for each
- [ ] Rankings applied all five criteria:
  - [ ] Algorithm Favorability (30%)
  - [ ] Trend Alignment (25%)
  - [ ] Niche Fit (20%)
  - [ ] Execution Feasibility (15%)
  - [ ] Differentiation Factor (10%)
- [ ] Rankings consider freshness vs previous batch

## Report Quality
- [ ] All sections populated
- [ ] Differentiation from previous batch explained
- [ ] Implementation strategy provided
- [ ] Refresh cadence recommendations included
- [ ] Quick wins highlighted
- [ ] Next steps are clear

## Differentiation Validation
- [ ] New ideas don't duplicate previous batch
- [ ] Fresh angles explored
- [ ] Variety in approach
- [ ] User has expanded options

**Validation Complete:** [ ] Report provides fresh, ranked ideas distinct from previous generation
