# Quick Niche Analysis - Validation Checklist

## Context Validation
- [ ] Niche is specific and clearly defined
- [ ] Analysis focus is documented (if specified)

## Niche Analysis Quality
- [ ] Data is from last 90 days maximum
- [ ] Instagram-specific audience behavior documented
- [ ] Viral mechanics identified for this niche
- [ ] Content saturation analysis included
- [ ] Fresh opportunities identified
- [ ] Algorithm considerations noted
- [ ] Data collection date displayed

## Competitor Landscape Quality
- [ ] Data is from last 30 days maximum
- [ ] At least 3-5 top accounts identified
- [ ] Common patterns across competitors documented
- [ ] Opportunity gaps clearly identified
- [ ] Data collection date displayed

## Report Quality
- [ ] All template sections populated
- [ ] Key takeaways are actionable
- [ ] Strategic recommendations provided
- [ ] Next steps are clear
- [ ] Report saved to correct location

## Actionability
- [ ] Insights can be immediately applied
- [ ] Opportunities are specific, not vague
- [ ] Recommendations are practical

**Validation Complete:** [ ] Report ready for delivery
