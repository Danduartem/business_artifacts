# Trend Report - Validation Checklist

## CRITICAL: Data Freshness Validation
- [ ] ALL data is from last 7 days ONLY (no older data included)
- [ ] Data collection date is prominently displayed
- [ ] Trend freshness warning is included
- [ ] Next refresh date is provided

## Trending Audio Quality
- [ ] 5-10 trending audio tracks identified
- [ ] Each audio track has name, artist, usage count estimate
- [ ] Audio usage patterns documented (which part, how it's used)
- [ ] Trend lifecycle noted (emerging, peaking, declining)
- [ ] Audio is from LAST 7 DAYS only

## Viral Format Quality
- [ ] Template formats identified with examples
- [ ] Hook patterns documented
- [ ] Text overlay styles noted
- [ ] Editing techniques specified
- [ ] Video length trends analyzed
- [ ] Formats are CURRENTLY trending (not outdated)

## Trending Effects Quality
- [ ] Trending effects identified by name
- [ ] Instagram features being leveraged documented
- [ ] Visual trends noted
- [ ] Examples of usage provided

## Breakout Content Analysis
- [ ] 3-5 recent breakout Reels identified
- [ ] Analysis of WHY each went viral
- [ ] Common patterns across breakouts noted
- [ ] All examples are from LAST 7 DAYS

## Report Quality
- [ ] All sections populated
- [ ] Trend freshness prominently emphasized
- [ ] Actionable strategy provided
- [ ] Timing considerations included
- [ ] Monitoring plan clear
- [ ] Warning about trend timing included

## Actionability
- [ ] User can immediately test identified trends
- [ ] Audio tracks are specific and findable
- [ ] Formats are clear and replicable
- [ ] Timing guidance is practical

## Data Integrity
- [ ] NO outdated trends included
- [ ] All trends verified as current
- [ ] Search queries used date filters
- [ ] Month/week specified in searches

**Validation Complete:** [ ] Report contains ONLY current trends from last 7 days
