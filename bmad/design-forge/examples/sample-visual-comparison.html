How to Generate Your Own Reports

To create visual comparison reports like this:

  1. Generate prototypes: *generate
  2. Capture screenshots: *screenshot (batch mode)
  3. Run analysis: *analyze (compare mode, include visuals: yes)
  4. Select HTML output format

Result:

  • Interactive HTML dashboard with real screenshots
  • Viewport switching (mobile/tablet/desktop)
  • Drag-to-compare image sliders
  • Complete design metrics
  • Lightbox viewing for enlarged screenshots
