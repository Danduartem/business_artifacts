# The Modern Artisan - Design Archetype Template

**Design Philosophy:** "Contemporary craft. Innovation meets refinement. Quality in every detail."

**Emotional Target:** Innovative, approachable, quality-conscious

**Visual Approach:** Modern techniques, balanced elements, refined details, subtle sophistication

## Core Parameters

### Layout Structure
- **Preferred:** Balanced asymmetry, modern grid systems, thoughtful spacing
- **Characteristics:** Contemporary, sophisticated, intentional
- **Philosophy:** Innovation within structure

### Visual Style/Mood
- **Core Style:** Modern, contemporary, balanced
- **Tone:** Innovative yet approachable, quality-focused
- **Aesthetic:** Swiss modernism meets digital craft

### Animation Patterns
- **Approach:** Smooth, refined, purposeful
- **Types:** Easing functions, smooth reveals, micro-interactions
- **Duration:** 250-350ms, custom easing curves
- **Philosophy:** Motion with meaning

### Color Schemes
- **Primary:** Contemporary palettes (teals, purples, modern blues)
- **Combinations:** Thoughtful, current but not trendy
- **Saturation:** Medium-high, refined
- **Approach:** Modern color theory application

### Typography Approach
- **Font Choice:** Geometric sans-serif, refined type systems
- **Hierarchy:** Modern scale, excellent rhythm
- **Scale:** Contemporary ratios (1.33 Golden Ratio)
- **Treatment:** Typographic excellence, refined details

### Spacing/Density
- **Philosophy:** Asymmetric but balanced
- **Approach:** Thoughtful negative space
- **System:** Modern spacing scale (4px, 8px, 16px, 24px, 32px)
- **Balance:** Neither minimal nor dense

### Content Arrangement
- **Alignment:** Balanced asymmetry, intentional off-center
- **Structure:** Modern grid with flexibility
- **Focus:** Clear but contemporary hierarchy
- **Philosophy:** Structure with creative freedom

### Visual Weight
- **Overall:** Balanced, contemporary
- **Elements:** Refined proportions, quality feel
- **Images:** High-quality photography, thoughtful crops
- **Approach:** Every element crafted with care

### Component Style
- **Buttons:** Modern design (subtle radius, refined hover)
- **Cards:** Contemporary shadows, clean designs
- **Forms:** Modern inputs, excellent UX
- **Navigation:** Sleek, functional, contemporary

### Interaction Patterns
- **Approach:** Refined, responsive, delightful
- **Hover:** Smooth transitions, subtle scale
- **Click:** Refined feedback, micro-animations
- **Philosophy:** Craft extends to interactions

## Best Practices

### Do:
- Use modern CSS techniques (clamp, Grid, custom properties)
- Balance asymmetry thoughtfully
- Craft refined micro-interactions
- Apply contemporary color palettes
- Use geometric sans-serif fonts
- Create sophisticated spacing systems

### Don't:
- Follow trends blindly
- Sacrifice usability for innovation
- Over-complicate simple tasks
- Ignore typographic principles
- Rush the details
- Choose flash over substance
