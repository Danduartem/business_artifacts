# The Minimalist - Design Archetype Template

**Design Philosophy:** "Less is more. Every element must earn its place. Clarity through reduction."

**Emotional Target:** Calm, sophisticated, clear

**Visual Approach:** Ample whitespace, restrained color palette (1-2 accent colors max), clear hierarchy, generous spacing

## Core Parameters

### Layout Structure
- **Preferred:** Centered, grid-based, spacious
- **Characteristics:** Clean alignment, predictable structure, breathing room between elements
- **Avoid:** Cluttered layouts, overlapping elements, complex asymmetry

### Visual Style/Mood
- **Core Style:** Minimal, clean, sophisticated
- **Tone:** Calm, professional, refined
- **Aesthetic:** Swiss design influence, modernist principles

### Animation Patterns
- **Approach:** Subtle, purposeful
- **Types:** Gentle fades, smooth transitions, micro-interactions
- **Duration:** 200-300ms, easing functions
- **Avoid:** Flashy animations, complex choreography, attention-seeking effects

### Color Schemes
- **Primary:** Monochrome base (whites, grays, blacks)
- **Accent:** 1-2 carefully chosen accent colors
- **Saturation:** Low to medium
- **Contrast:** High for readability, subtle for hierarchy

### Typography Approach
- **Font Choice:** Clean sans-serif (Inter, Helvetica, SF Pro)
- **Hierarchy:** Clear size differences, generous line height
- **Scale:** Modular scale (1.25 or 1.33 ratio)
- **Treatment:** Excellent readability, no decorative fonts

### Spacing/Density
- **Philosophy:** Airy, spacious, generous
- **Margins:** Large (24-48px typical)
- **Padding:** Ample breathing room
- **White Space:** Active design element, not leftover space

### Content Arrangement
- **Alignment:** Centered or left-aligned (never right-aligned)
- **Structure:** Clear visual grouping
- **Focus:** One primary element per section
- **Avoid:** Multiple competing focal points

### Visual Weight
- **Overall:** Minimal, light
- **Elements:** Only essential components
- **Images:** Clean product shots, simple graphics
- **Avoid:** Heavy textures, busy patterns, excessive decoration

### Component Style
- **Buttons:** Simple borders or subtle fills, clear labels
- **Cards:** Clean borders or subtle shadows, never cluttered
- **Forms:** Minimal styling, clear focus states
- **Navigation:** Simple, unobtrusive

### Interaction Patterns
- **Approach:** Subtle, responsive
- **Hover:** Gentle color shifts, underlines
- **Click:** Clear feedback without drama
- **States:** Obvious but not flashy

## Best Practices

### Do:
- Use generous whitespace as an active design element
- Maintain strict visual hierarchy
- Choose 1-2 accent colors maximum
- Ensure every element has clear purpose
- Use clean, readable typography
- Create breathing room between sections

### Don't:
- Overcomplicate with unnecessary elements
- Use decorative flourishes without purpose
- Ignore whitespace opportunities
- Mix multiple visual styles
- Add "just one more thing"
- Sacrifice clarity for creativity

## Example Sections

**Minimalist Hero:**
- Large headline (60-80px)
- Simple subheadline (18-24px)
- Single CTA button (minimal style)
- Clean product image or nothing
- Massive top/bottom padding (80-120px)
- Centered alignment

**Minimalist Features:**
- 3 features maximum per row
- Icon + headline + short description
- Consistent spacing grid
- Monochrome icons
- No backgrounds, just clean layout

**Minimalist Pricing:**
- 3 columns with equal visual weight
- Simple borders, no shadows
- Clear price emphasis
- Minimal feature lists (5-7 items max)
- One subtle accent for recommended plan
