# The Elegant Curator - Design Archetype Template

**Design Philosophy:** "Luxury through restraint. Excellence in the details. Sophistication speaks softly."

**Emotional Target:** Elegant, exclusive, aspirational

**Visual Approach:** Sophisticated typography, subtle animations, premium feel, refined color palettes

## Core Parameters

### Layout Structure
- **Preferred:** Spacious, elegant proportions, carefully curated elements
- **Characteristics:** Refined, luxurious, uncluttered
- **Philosophy:** Curation over abundance

### Visual Style/Mood
- **Core Style:** Elegant, refined, luxurious
- **Tone:** Exclusive, aspirational, sophisticated
- **Aesthetic:** High-end editorial, luxury brand aesthetic

### Animation Patterns
- **Approach:** Refined, subtle, luxurious
- **Types:** Gentle fades, elegant reveals, sophisticated transitions
- **Duration:** 350-500ms, elegant easing curves
- **Philosophy:** Motion should feel expensive

### Color Schemes
- **Primary:** Refined palettes (deep blues, rich burgundies, elegant neutrals)
- **Combinations:** Sophisticated, often muted or monochromatic
- **Saturation:** Low to medium, elegant restraint
- **Approach:** Color as accent, never overwhelming

### Typography Approach
- **Font Choice:** Refined serifs, elegant sans-serif pairings
- **Hierarchy:** Sophisticated scale, excellent typographic rhythm
- **Scale:** Elegant ratios, generous leading
- **Treatment:** Typography as art, refined details (ligatures, kerning)

### Spacing/Density
- **Philosophy:** Spacious, elegant, generous
- **Approach:** Luxury through space
- **Breathing Room:** Essential to elegance
- **Balance:** More space than you think necessary

### Content Arrangement
- **Alignment:** Spacious, elegant proportions
- **Structure:** Carefully curated, every element intentional
- **Focus:** Few elements, each exquisite
- **Philosophy:** Less is luxury

### Visual Weight
- **Overall:** Refined, never heavy
- **Elements:** Carefully chosen, impeccably executed
- **Images:** High-end photography, elegant compositions
- **Approach:** Quality over quantity always

### Component Style
- **Buttons:** Refined, often outlined or subtle
- **Cards:** Elegant borders, refined shadows or none
- **Forms:** Sophisticated styling, luxury UX
- **Navigation:** Understated, refined, minimal

### Interaction Patterns
- **Approach:** Subtle, sophisticated, refined
- **Hover:** Gentle reveals, elegant transitions
- **Click:** Refined feedback, luxurious feel
- **Philosophy:** Interactions should feel premium

## Best Practices

### Do:
- Use refined serif typography
- Create generous spacing
- Choose sophisticated color palettes
- Curate elements carefully (remove more than you add)
- Pay attention to typographic details
- Create elegant proportions

### Don't:
- Overcrowd the design
- Use bright, loud colors
- Rush transitions or interactions
- Add elements without purpose
- Ignore typographic refinement
- Sacrifice elegance for trends
