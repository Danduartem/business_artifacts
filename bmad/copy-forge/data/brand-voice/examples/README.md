## Which Format Should You Use?

**Use Comprehensive Format If:**
- You manage multiple brands/clients
- You need institutional-grade documentation
- You're building a brand style guide
- You have large teams needing alignment
- You value extreme precision

**Use Simplified Format If:**
- You're a solo entrepreneur
- You want to start using Copy Forge quickly
- You're documenting your own brand
- You prefer practical over exhaustive
- You can always add detail later

**Good News:** Copy Forge's Brand Voice Guardian works with BOTH formats. The validator adapts to whatever level of detail you provide.
