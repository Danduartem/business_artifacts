# Forge Complete LP - Quality Checklist

## Pre-Generation Checklist

- [ ] LP blueprint provided and parsed
- [ ] Brand voice document loaded
- [ ] Avatar profile loaded
- [ ] Blueprint interpretation confirmed by user

## Per-Section Checklist

### Copywriting
- [ ] Creative brief created with section details
- [ ] Appropriate copywriter assigned (Hero Writer or CTA Specialist)
- [ ] Word count target specified
- [ ] Brand voice and avatar context provided

### Brand Voice Validation (7/7 required)
- [ ] Tone alignment - matches brand guidelines
- [ ] Vocabulary choices - uses brand terms, avoids forbidden words
- [ ] Sentence rhythm - matches brand writing style
- [ ] Personality expression - brand character shines through
- [ ] Messaging consistency - aligns with core messages
- [ ] Authenticity - genuinely sounds like the brand
- [ ] Emotional resonance - matches brand emotional tone

### Audience Resonance Validation (8/8 required)
- [ ] Language match - uses avatar's words
- [ ] Pain point resonance - avatar feels seen
- [ ] Desire alignment - speaks to what avatar wants
- [ ] Objection handling - concerns addressed
- [ ] Clarity level - appropriate sophistication
- [ ] Relatability - written FOR them specifically
- [ ] Emotional connection - triggers right emotions
- [ ] Trust building - addresses skepticism

## Cohesion Review Checklist

- [ ] Flow between sections - smooth transitions
- [ ] Message consistency - core message reinforced
- [ ] Tone maintenance - brand voice consistent throughout
- [ ] Avatar journey - emotional arc makes sense
- [ ] Redundancy check - no unnecessary repetition
- [ ] CTA placement - appears at right moments
- [ ] Structure logic - sequence makes strategic sense

## Output Checklist

- [ ] {brand}-landing-page.md compiled
- [ ] {brand}-validation-report.md compiled
- [ ] All sections included in correct order
- [ ] Total word count verified
- [ ] Overall scores calculated and presented
- [ ] Refinement options offered to user
