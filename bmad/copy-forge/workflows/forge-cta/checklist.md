# Forge CTA - Quality Checklist

## Pre-Generation
- [ ] Brand voice document loaded
- [ ] Avatar profile loaded
- [ ] Desired action clarified
- [ ] Urgency level determined (genuine or ongoing)
- [ ] Conversion context understood

## CTA Creation
- [ ] Friction points identified from avatar profile
- [ ] Avatar objections and motivations analyzed
- [ ] 3-5 button text variations created
- [ ] Strongest button text selected with rationale
- [ ] Supporting microcopy reduces friction
- [ ] Urgency is GENUINE (or absent)
- [ ] Alternative CTA for hesitant users (if needed)

## Friction Reduction
- [ ] Decision friction addressed (clear single action)
- [ ] Risk friction addressed (guarantee, trial)
- [ ] Time friction addressed (genuine urgency only)
- [ ] Complexity friction addressed (clear next step)
- [ ] Trust friction addressed (social proof)

## Brand Voice Validation (7/7)
- [ ] Tone alignment
- [ ] Vocabulary choices
- [ ] Sentence rhythm
- [ ] Personality expression
- [ ] Messaging consistency
- [ ] Authenticity
- [ ] Emotional resonance

## Audience Resonance Validation (8/8)
- [ ] Language match
- [ ] Pain point resonance
- [ ] Desire alignment
- [ ] Objection handling
- [ ] Clarity level
- [ ] Relatability
- [ ] Emotional connection
- [ ] Trust building

## Urgency Check
- [ ] If urgency used, it is GENUINE
- [ ] No fake countdown timers
- [ ] No false scarcity claims
- [ ] Deadline/capacity is real

## Output
- [ ] CTA section formatted correctly
- [ ] Validation scores included
- [ ] Strategic rationale provided
- [ ] Refinement options offered
