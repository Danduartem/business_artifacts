# Compare Personas Workflow - Validation Checklist

- [ ] Correct number of personas loaded (2-3)
- [ ] All persona files successfully parsed

### Comparison Generation
- [ ] Side-by-side comparison generated for all requested dimensions
- [ ] Comparison tables formatted correctly
- [ ] Key differences highlighted
- [ ] Similarities identified
- [ ] Overlap risks noted

### Analysis Quality
- [ ] Demographics comparison complete and accurate

- [ ] Pain points and goals compared
- [ ] Awareness journey differences documented
- [ ] Qualification question differentiation analyzed

### Strategic Insights (if strategic comparison)
- [ ] Market opportunity analysis conducted
- [ ] Resource allocation recommendations provided
- [ ] Cross-persona insights identified
- [ ] Strategic recommendations actionable and specific
- [ ] Go-to-market strategy outlined

### Specialist Review (if requested)
- [ ] Senior Persona Specialist review completed
- [ ] Strategic validation provided
- [ ] Additional opportunities identified

### Action Recommendations
- [ ] Immediate actions listed (3-5 items)

- [ ] Long-term strategy outlined
- [ ] Differentiation tactics specific
- [ ] Resource allocation recommendations clear

### Report Generation
- [ ] Comparison report document generated
- [ ] Executive summary included

- [ ] Next steps clearly communicated

## Quality Standards

### Comparison Accuracy
- [ ] All persona attributes compared correctly
- [ ] No misattribution of characteristics
- [ ] Differences accurately identified
- [ ] Similarities correctly noted
- [ ] Analysis logically sound

### Insight Quality
- [ ] Insights are specific, not generic
- [ ] Recommendations are actionable
- [ ] Strategic advice is relevant to business context
- [ ] Opportunity analysis is realistic
- [ ] Risk assessment is thorough

### Report Usability
- [ ] Report is well-organized and scannable
- [ ] Tables and visual elements enhance understanding
- [ ] Executive summary provides quick overview
- [ ] Action items are prioritized
- [ ] Report can be shared with team
